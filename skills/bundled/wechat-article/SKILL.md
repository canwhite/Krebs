## 一、标题生成技巧

### 黄金标题公式

#### 1. 痛点 + 解决方案 + 数字诱惑
```
格式：[痛点人群]必看！这[N]个方法让你[解决痛点]

示例：
- "熬夜党必看！这5个方法让你第二天满血复活"
- "体检报告这5个指标异常，离癌症只差一步！"
- "35岁被裁员，只因你没看懂这3条职场潜规则"
```

#### 2. 好奇 + 悬念 + 反常识
```
格式：[A]和[B]，差的不是[表面因素]，而是[深层原因]

示例：
- "月薪3000和月薪3万的人，差的不是努力，而是这个习惯"
- "数据惊人：68%的10万+文章，竟然都是深度长文"
```

#### 3. 情感共鸣 + 场景化
```
格式：[年龄/身份]的你，有多久没[情感场景]了？

示例：
- "30岁的你，有多久没好好睡一觉了？"
- "成年人的崩溃，往往从借不到500块钱开始"
```

### 标题生成步骤

1. **明确目标人群**（如：职场人、宝妈、学生）
2. **挖掘核心痛点**（如：焦虑、疲惫、迷茫）
3. **提供数字解决方案**（3个方法、5个步骤、7天改变）
4. **添加情绪词**（必看、惊人、警惕）

## 二、开头钩子技巧（前200字决定生死）

### 1. 数据破冰法（制造认知冲突）
```
格式：打破认知的惊人数据 + 引出话题

示例：
"但数据 shows，2023年68%的10万+文章是深度长文，
平均阅读时间反而增加了17%。在碎片化时代，用户
更愿意为深度内容买单。"
```

### 2. 问题聚焦法（引发思考）
```
格式：直击核心问题 + 制造悬念

示例：
"为什么在信息爆炸的时代，我们反而越来越难静下
心来读完一篇长文？"
```

### 3. 场景代入法（拉近距离）
```
格式：真实场景描述 + 让读者"看见自己"

示例：
"凌晨2点，你躺在床上刷着手机，屏幕的光打在脸上。
朋友圈里别人都在晒加班、晒学习，你突然一阵
焦虑：我是不是又虚度了一天？"
```

### 4. 故事引入法（情感共鸣）
```
格式：具体人物故事 + 情绪转折点

示例：
"上周我的朋友小林在聚会上哭了。35岁，两个孩子的
妈妈，外企中层，在别人眼里是人生赢家。但她说：
'我每天像上了发条的机器，不知道为什么而活。'"
```

### 开头钩子检查清单

- [ ] 是否在前3句话抓住注意力？
- [ ] 是否制造了认知冲突或悬念？
- [ ] 是否让读者"看见自己"？
- [ ] 是否引发情感共鸣？
- [ ] 是否自然过渡到正文？

### 第一部分：痛点放大（情绪钩子）

**目标**：让读者觉得"懂我"、"在说我"

#### 7大情绪驱动力（基于200篇10万+爆文复盘）

| 情绪类型 | 使用场景 | 示例句式 |
|---------|---------|---------|
| **不甘心** | 职场、成长 | "明明很努力，为什么还是原地踏步？" |
| **羞辱感** | 社交、身份 | "成年人的崩溃，往往从借不到500块开始" |
| **疲惫崩溃** | 健康、压力 | "35岁的身体，已经经不起熬夜了" |
| **想证明自己** | 能力、价值 | "我想让那些看轻我的人闭嘴" |
| **对未来的焦虑** | 职业、家庭 | "30岁了，我还在为下个月的房贷发愁" |
| **想被看到** | 存在感、认同 | "在朋友圈，为什么没人点赞我的生活？" |
| **孤独与坚强** | 情感、独立 | "成年人的世界，连崩溃都要选时间" |

#### 痛点挖掘三维度

**维度1：生存焦虑**
- 健康：体检异常、癌症恐惧、亚健康
- 职业：裁员、35岁危机、薪资瓶颈
- 经济：房贷压力、失业恐惧、养老焦虑

**维度2：社交焦虑**
- 身份认同："我是谁"、"我在哪"
- 群体归属："我是不是被时代抛弃了"
- 比较陷阱："为什么同龄人比我成功"

**维度3：效率焦虑**
- 时间成本："时间都去哪了"
- 工作效率："为什么我总是拖延"
- 学习焦虑："知识焦虑症候群"

### 第二部分：案例论证（细节+数据）

**目标**：用真实感增强说服力

#### 案例选择原则

1. **具体化**：有具体人物、场景、数字
2. **可感知**：让读者"能看见"、"能触摸"
3. **反差感**：前后对比强烈
4. **代表性**：读者能代入自己

#### 案例写作模板

```
【人物背景】
我的朋友/同事/读者小X，[年龄]，[职业/身份]，[现状描述]

【问题出现】
[具体问题发生的时间、场景]
"直接引用原话，增强真实感"

【尝试解决】
尝试了[方法A]、[方法B]，但都[失败原因]

【转折点】
直到[某个契机/发现]，才找到[根本原因/解决方法]

【结果改变】
现在[具体改善数据]，整个人[状态变化]
```

#### 数据使用技巧

1. **数据破冰**：用反常识数据打破认知
2. **对比数据**：前后对比、行业对比
3. **权威背书**：引用权威机构数据
4. **具体化数字**："68%"比"大多数"更有说服力

### 第三部分：解决方案（干货输出）

**目标**：提供简单可执行的方法

#### 方法设计原则

1. **数字化**：3个方法、5个步骤、7天计划
2. **简单化**：降低执行门槛
3. **可操作**：具体步骤，不是空泛道理
4. **递进式**：从易到难，循序渐进

#### 方法写作模板

```
【方法标题】[动词+名词]
[一句话概括这个方法的作用]

具体步骤：
1. [第一步]：[具体操作]
2. [第二步]：[具体操作]
3. [第三步]：[具体操作]

💡 执行贴士：
- [注意事项1]
- [注意事项2]

真实效果：
[案例佐证：谁用了，效果如何]
```

### 第四部分：情绪升华（治愈力量）

**目标**：给予治愈感或价值认同

#### 升华方式

1. **治愈式升华**：温暖话语，给予力量
2. **认同式升华**：价值共鸣，确认"你值得"
3. **行动式升华**：鼓励改变，从今天开始
4. **金句式升华**：提炼核心观点，易于传播

#### 金句创作技巧

**公式1：对比+转折**
```
"真正的成长，不是追求完美，而是接受不完美的自己。"
```

**公式2：时间+意义**
```
"人生最好的投资，是把时间花在能复利的事情上。"
```

**公式3：否定+肯定**
```
"不要让你的焦虑，毁了你本可以拥有的幸福。"
```

## 七、实战流程

当用户要求创作公众号文章时，按以下流程执行：

### Step 1: 明确需求
询问用户：
1. **文章主题**（如：职场成长、健康生活、情感关系）
2. **目标人群**（如：25-35岁职场女性）
3. **核心痛点**（如：职业焦虑、时间管理）
4. **期望情绪**（如：治愈、激励、警示）
5. **文章长度**（默认2000字左右）

### Step 2: 生成标题
基于用户需求，提供3-5个标题供选择，每个标题都符合爆款公式。

### Step 3: 生成大纲
创建文章大纲，明确：
- 开头钩子类型（数据破冰/问题聚焦/场景代入/故事引入）
- 痛点放大方向（7大情绪驱动力）
- 案例选择（具体人物+场景+数据）
- 解决方案（3-5个方法）
- 情绪升华方式（治愈/认同/行动/金句）

### Step 4: 撰写正文
按照四段式结构撰写：
1. 痛点放大（20%）- 引发焦虑
2. 案例论证（30%）- 增强可信度
3. 解决方案（30%）- 提供方法
4. 情绪升华（20%）- 治愈力量

### Step 5: 优化结尾
添加：
- 互动引导（提问/共鸣/行动）
- 行动召唤（订阅/转发/服务）
- 价值强化（总结要点/收藏理由）

### Step 6: 质量检查
对照写作检查清单，确保满足所有标准。
