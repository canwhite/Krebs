# 微信公众号文章写作 Skill 测试

## 测试场景 1：职场焦虑文章

### 用户输入
```
帮我写一篇关于"职场焦虑"的公众号文章，目标读者是28-35岁的互联网从业者
```

### 预期输出
Skill 应该：
1. 生成3-5个爆款标题供选择
2. 提供完整的文章大纲
3. 按照四段式结构撰写全文（约2000字）
4. 包含开头钩子、痛点放大、案例论证、解决方案、情绪升华
5. 结尾有互动引导和行动召唤

### 关键验证点
- [ ] 标题符合"痛点+解决方案+数字"公式
- [ ] 开头前3句话抓住注意力
- [ ] 痛点放大使用了7大情绪驱动力之一
- [ ] 有真实可感的案例
- [ ] 提供3-5个具体方法
- [ ] 有金句收尾
- [ ] 结尾有互动引导

## 测试场景 2：健康生活文章

### 用户输入
```
写一篇关于"熬夜危害"的公众号文章，目标是年轻人
```

### 预期输出
1. 标题示例："熬夜党必看！这5个方法让你第二天满血复活"
2. 开头使用场景代入："凌晨2点，你躺在床上刷着手机..."
3. 痛点放大：健康焦虑、生存焦虑
4. 案例论证：真实案例+体检数据
5. 解决方案：5个改善睡眠的方法
6. 情绪升华：治愈式结尾

## 测试场景 3：情感关系文章

### 用户输入
```
写一篇关于"成年人社交"的公众号文章
```

### 预期输出
1. 标题示例："成年人的崩溃，往往从借不到500块钱开始"
2. 开头使用故事引入
3. 痛点放大：孤独感、社交焦虑
4. 案例论证：真实故事
5. 解决方案：建立边界感、质量优先于数量
6. 情绪升华：金句"真正的成熟，是知世故而不世故"
